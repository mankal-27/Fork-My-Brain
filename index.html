Adding a new module

step 01
Create a folder
Each topic lives in its own folder at the repo root.
Fork-My-Brain/ ├── js-array-guide/ ├── js-string-guide/ ← new └── index.html
step 02
Copy the template
Duplicate js-array-guide/ and replace the scenario data in js/scenarios.js.
cp -r js-array-guide \ js-string-guide
step 03
Add the card here
Add a new .module-card in index.html and set status to Live.
<!-- status: live / wip / soon --> <span class="mc-status live"> Live </span>
step 04
Push & deploy
GitHub Pages auto-deploys from main. No build step needed — pure HTML/CSS/JS.
git add . git commit -m "add string guide" git push
